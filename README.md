🚀 Calculator Game

A fun, interactive calculator game built with HTML, CSS, and JavaScript.
Solve three hidden number challenges using clues unlocked through the calculator interface — and celebrate with popups and confetti when you win!

🎮 How to Play
Use the calculator normally.
Click the I, II, and III buttons to reveal hints.
Each hint corresponds to one specific number.
Solve the hint by calculating the correct number.
When you're correct, the achievement button turns green.
Solve all 3 hints to win — and unlock a confetti celebration!


🧩 Features
Fully functional calculator
Error handling
Hint system with auto-clear
Calculation history displayed in the background
Achievement buttons that change color when solved
Rules popup on load
Win popup
Confetti celebration animation
Clean UI styled with CSS
100% front-end — no backend required

🛠️ Technologies Used
HTML5 – layout & structure
CSS3 – styling & layout
JavaScript – logic & interactivity
canvas-confetti – celebration effect

🎉 Winning Condition
To win the game: 
Correctly solve all 3 hints
Turn all achievement buttons green
Trigger the “Congratulations!” popup
Enjoy the confetti celebration

📦 Installation & Setup
Clone the repository:
git clone https://github.com/your-username/Calculator-Game.git
Open index.html in any browser.
No installation or build tools needed.
